# Login System

A simple Flask-based login system with user registration and authentication.

## Setup

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Run the application:
   ```
   python app.py
   ```

3. Open your browser and go to `http://127.0.0.1:5000/`

## Features

- User registration
- User login
- Password hashing
- Session management
- Basic dashboard

## Database

The application uses SQLite database (`users.db`) to store user credentials. The database is initialized automatically when the app starts.
